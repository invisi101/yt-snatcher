# YT Snatcher

![YT Snatcher](icons/yt-snatcher-logo.svg)

A PyQt5 GUI for downloading audio and video from YouTube using yt-dlp.

## Install

```bash
git clone https://github.com/invisi101/yt-snatcher.git
cd yt-snatcher
./install.sh
```

This installs `yt-dlp` and `python3 PyQt5` via your package manager (supports pacman, dnf, and apt), then copies the app, icon, and desktop entry to `~/.local/`. You can delete the cloned `yt-snatcher` folder after installation.

## Usage

Launch from your app menu, or:

```bash
yt-snatcher
```

## Uninstall

```bash
./uninstall.sh
```
